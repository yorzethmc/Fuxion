# Documentacion del proyecto

Esta carpeta documenta el proyecto **Fusion Culinaria - Catalogo de Servicios Web**.

El objetivo es dejar claro:

- Que esta construido.
- Con que tecnologias esta construido.
- Como se ejecuta cada demo.
- Donde se cambian datos importantes como menu, precios y WhatsApp.
- Que falta o que conviene mejorar antes de usarlo en produccion.

## Archivos

- [01-resumen-del-proyecto.md](01-resumen-del-proyecto.md): vision general del proyecto.
- [02-arquitectura-y-tecnologias.md](02-arquitectura-y-tecnologias.md): estructura, tecnologias y patrones usados.
- [03-guia-de-ejecucion.md](03-guia-de-ejecucion.md): instrucciones para abrir o correr cada tipo de demo.
- [04-rangos-y-categorias.md](04-rangos-y-categorias.md): detalle de cada rango y categoria.
- [05-datos-y-configuracion.md](05-datos-y-configuracion.md): datos del negocio, menu, precios y configuraciones.
- [06-pendientes-y-mejoras.md](06-pendientes-y-mejoras.md): faltantes, riesgos y mejoras recomendadas.
- [07-raw-data-y-fuentes.md](07-raw-data-y-fuentes.md): archivos que funcionan como fuente de informacion del proyecto.
