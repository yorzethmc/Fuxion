# Resumen del proyecto

## Que es

Este repositorio contiene un catalogo de demos web para restaurantes, usando como ejemplo a **Fusion Culinaria**, ubicado en Sabanilla, Alajuela.

La idea principal es mostrar diferentes niveles de servicio digital para un negocio de comida:

- Menu basico mobile-first.
- Menu QR con botones de ubicacion y pedidos.
- Menu editable mediante archivo JSON.
- Sitio profesional en React.
- Pre-pedido por WhatsApp con carrito.
- Combo de marca con SEO local y configuracion de despliegue.
- Experiencia premium visual con imagenes y secciones de marca.
- E-commerce gourmet con carrito, checkout y validaciones.
- Cloud Kitchen & POS con menu cliente, admin, cocina y datos demo.

## Estado actual

El proyecto ya tiene una estructura funcional de demos separadas por rangos:

- `RANGO_1_ECONOMICO`: demos HTML simples.
- `RANGO_2_PROFESIONAL`: demos Vite + React.
- `RANGO_3_PREMIUM`: demos Vite + React con tratamiento visual premium, e-commerce y operacion Cloud Kitchen/POS.

Todas las demos estan orientadas a celular y usan WhatsApp como canal principal para convertir visitas en pedidos.

## Publico objetivo

El proyecto parece pensado para vender o presentar paquetes web a restaurantes pequenos o negocios de comida que necesitan presencia digital rapida, menu visible y pedidos directos sin construir un sistema completo de ecommerce.
